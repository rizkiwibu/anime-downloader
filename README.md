# 🎌 Anime Downloader Bot – Bot Telegram untuk Download Anime! 🎌  

**Anime Downloader Bot** adalah bot Telegram yang memungkinkan kamu mencari dan mengunduh anime favorit langsung dari Telegram dengan mudah!  

🔹 **Mudah digunakan** – cukup ketik perintah dan dapatkan hasilnya!  
🔹 **Selalu update** – anime terbaru langsung tersedia!  
🔹 **Didukung oleh REST API** dari [restapi.botwaaa.web.id](https://restapi.botwaaa.web.id)  

---

## ✨ Fitur Utama  

### 🔍 Mencari Anime  
Cari anime favoritmu berdasarkan judul dengan perintah:  
```bash
/search <judul anime>
```
Contoh:  
```bash
/search One Piece
```

### 📺 Update Anime Terbaru  
Lihat daftar anime terbaru yang telah dirilis dengan perintah:  
```bash
/latest
```

### 🎬 Detail Anime  
Dapatkan informasi lengkap mengenai anime tertentu dengan perintah:  
```bash
/detail <url anime>
```
Contoh:  
```bash
/detail https://example.com/anime/one-piece
```

### ⬇️ Download Anime  
Dapatkan link download episode anime favoritmu dengan perintah:  
```bash
/download <url episode>
```
Contoh:  
```bash
/download https://example.com/episode/one-piece-1000
```

## 📥 Cara Install  

Bot ini bersifat **Open Source**, kamu bisa menjalankannya sendiri dengan langkah-langkah berikut:  

```bash
git clone https://github.com/rizkiwibu/anime-downloader
cd anime-downloader
npm install
npm start
```  

📝 **Jangan lupa untuk mengisi file `.env` sebelum menjalankan bot!**
